# 🛒 Amazon Clone (Static Frontend)

A responsive static clone of the Amazon homepage built using HTML and CSS.
This project focuses on replicating UI/UX design and layout structure.

---

## 🚀 Features

* Fully responsive design
* Navbar with search bar
* Product sections layout
* Footer similar to Amazon
* Clean UI structure

---

## 🛠️ Tech Stack

* HTML5
* CSS3
